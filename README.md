## Hybrid Retarded Clock
A futuristic hybrid clock built using HTML, CSS, and Js.
Combines the aesthetics of an analog clock with the functionality of digital time display.
Each hand rotates like an analog clock while displaying the current time value as a glowing digital number.

## Features

- Analog-style circular clock face with hour marks and ticks
- Rotating hour, minute, and second hands showing digital numbers
- Smooth animations using `requestAnimationFrame`
- Neon/glowing effects for hands and digits
- Dark theme with gradient or glassmorphic background
- Responsive design for different screen sizes
- Optional: dynamic color changes based on time

## Tech Stack
- HTML5 Canvas / SVG
- CSS3
- Vanilla JavaScript


## Usage
1. Clone the repository or download the files
2. Open `index.html` in a web browser
3. Watch the hybrid clock in action!
